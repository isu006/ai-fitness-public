## How to Capture Screenshots

### Using Your Live Application

1. Navigate to: https://ai-fitness-4l4lr3cooq-uc.a.run.app
2. Log in to your account
3. Use Windows Snipping Tool (Win + Shift + S) or Snip & Sketch

### Recommended Tools for GIFs

- **ScreenToGif** (Windows, free): https://www.screentogif.com/
- **LICEcap** (Windows/Mac, free): https://www.cockos.com/licecap/
- **Kap** (Mac, free): https://getkap.co/

### GIF Recording Settings

- **Resolution**: 1280x720 (720p) or 1920x1080 (1080p)
- **Frame Rate**: 15-20 FPS
- **Duration**: 10-15 seconds per demo
- **File Size**: Aim for under 5MB per GIF (use compression)

## Image Optimization

Before committing images to the repository:

1. **Compress PNG files**: Use TinyPNG (https://tinypng.com/)
2. **Optimize GIFs**: Use ezgif.com or gifsicle
3. **Target file sizes**:
   - Screenshots: < 500KB each
   - Demo GIFs: < 5MB each
   - Architecture diagrams: < 1MB each
